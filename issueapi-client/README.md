# An API client for IssueAPI

This is a simple API client for IssueAPI. The code has been made based on the learnings of the [Full Stack Open course](https://fullstackopen.com/). There are similar structures and patterns used as in the course materials.

## Prerequisites

- Node.js 18 or higher
- npm 9 or higher

## Dependencies

This project relies on the following key technologies and libraries:

### Core UI & Framework
- **React**
- **React Router**

### Data Fetching & State Management
- **TanStack Query**
- **Zustand**

### Styling
- **Tailwind CSS**

## How to install dependencies and run the client in development mode

```bash
cd issueapi-client
npm install
npm run dev
```

Make sure that the backend is running on port 5000.

## How to run build, test and linting

There are provided npm scripts for these tasks. All commands can be run in the `issueapi-client` directory.

### Running build

```bash
npm run build
```

### Running linting

```bash
npm run lint
```

### Running tests

```bash
npm run test
```
**Note:** Because time limits, there are not test. Just testing environment setuped and there is one sample test. (edited: 3.5.2026).
